This project is a simple one and it's my exploration into the [Three.js](https://threejs.org) world.

# Getting Started

1. Clone the project
2. Make sure you have [pnpm](https://pnpm.io) installed
3. Run
```bash
  pnpm install
```
To install dependencies
4. Then Run
```bash
  pnpm run dev
```
To start the project
